# SmartFin Pro — Full-Stack Personal Finance Manager

A responsive fintech web application for income/expense tracking, budgets, savings goals and financial analytics.

## Tech stack
React + Vite, JavaScript, CSS, Chart.js, Axios, Node.js, Express, JWT, bcrypt, JSON persistence.

## Features
- Secure signup/login with encrypted passwords and JWT
- Responsive dashboard for mobile, tablet and desktop
- Income and expense CRUD with search and filtering
- Interactive income-vs-expense and category charts
- Monthly category budgets with live progress
- Savings goals with contribution tracking
- Persistent local database, validation and protected APIs

## Run on Windows
1. Install Node.js LTS and restart VS Code.
2. Open this project folder in VS Code.
3. Run:

```bash
npm install
npm run install:all
npm run dev
```

Open http://localhost:5173. API runs at http://localhost:5000/api.

The `.env` files are already included, so no copying is required.
